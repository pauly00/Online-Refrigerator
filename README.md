
# 🧊 Online Refrigerator

**식재료 관리를 위한 온라인 냉장고**

> 2023-2 동국대학교 웹프로그래밍 전공 팀 프로젝트

사용자의 냉장고 속 식재료를 디지털로 관리하고, 유통기한이 임박한 재료를 활용한 레시피를 추천하여 식생활의 편의성과 효율성을 높이는 웹 서비스입니다.
재고 관리, 레시피 추천, 식비 관리 기능을 하나의 서비스로 통합하였습니다.

---

## 💡 Core Features

### 1. 스마트 냉장고 관리 (Inventory Management)

**식재료 등록**

* CSV 데이터와 연동된 검색 기능 제공
* 재료 선택 후 가격, 수량, 유통기한 입력 및 저장

**재료 현황 확인**

* 냉장고 UI를 통해 보유 식재료 목록 및 상세 정보 확인
* 유통기한 기준 정렬 및 상태 확인 가능

**실시간 데이터 반영**

* 재료 삭제 시 보유 목록 및 레시피 추천 결과에 즉시 반영
* DB와 연동된 동적 갱신 처리

---

### 2. 지능형 레시피 추천 (Recipe Recommendation)

**공공 데이터 API 연동**

* 식약처 조리식품 레시피 공공 API 활용
* 신뢰도 높은 외부 데이터 실시간 파싱

**맞춤형 레시피 제공**

* 현재 냉장고에 존재하는 식재료 조합 분석
* 즉시 조리 가능한 메뉴를 우선 추천
* 보유 재료 기반 필터링 로직 구현

---

### 3. 식비 가계부 및 알림 기능 (Household Account Book)

**지출 시각화**

* 캘린더 UI 기반 일자별 식비 조회
* 월간 총 지출 자동 계산

**유통기한 알림**

* 임박 재료 리스트 제공
* 식재료 낭비 최소화를 위한 시각적 안내

---
